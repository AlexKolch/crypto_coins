# crypto_coins

Crypto market platform.

Receive data about crypto market. 

Used in a project:

* Flutter
* Dio
* REST API, JSON (json_serializable) 
* Dependency injection (Get_It)
* Bloc, Equatable
* Hive
* AutoRoute
